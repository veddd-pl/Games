Here's a full reference of the standard controls/keybinds for **Gunner, HEAT, PC!** (GHPC):

> **Tip:** You can press **F1** during any mission to show the in-game controls list at any time.

---

### 🚗 Movement (Driver)


| Key | Action |
|-----|--------|
| **W / A / S / D** | Drive forward / turn left / reverse / turn right |
| **Numpad +** | Increase cruise speed |
| **Numpad −** | Decrease cruise speed |

All tanks shift gears automatically.

---

### 🎯 Gunnery & Sights


| Key | Action |
|-----|--------|
| **Shift** | Enter gun sight view |
| **Right Mouse Button (hold)** | Aim weapon system (in gun sight) |
| **Left Mouse Button** | Fire |
| **[ ]** | Cycle available weapon systems |



| Key | Action |
|-----|--------|
| **1, 2, 3...** | Select ammo type (number keys along top of keyboard) |
| **Left Ctrl + Scroll Wheel** | Adjust range (also PgUp / PgDn) |
| **PgUp / PgDn** | Adjust range up/down |
| **Ctrl + E** | Cancel delta-D (automatic range adjustment) |

---

### 👁️ Camera & Views


| Key | Action |
|-----|--------|
| **Shift** | Gunner's sight view |
| **Right-click (exterior view)** | Direct gunner aim |
| **Scroll Wheel (commander view)** | Zoom in/out |



| Key | Action |
|-----|--------|
| **F3** | Toggle free cam (in exterior view) |
| **C** | Gunner auxiliary sight (M60/M1 series only) |

---

### 🔭 Free Cam Controls


| Key | Action |
|-----|--------|
| **W / A / S / D** | Move (level with horizon) |
| **Space / Ctrl** | Rise / Sink |
| **Scroll Wheel** | Flight speed |
| **Shift + Scroll Wheel** | Field of view |
| **X** | Reset FOV |
| **Shift + Left Click** | Fire current weapon |
| **Hold Right Click** | Smooth pan and FOV |



---

### 🔧 Systems & Quick Menu
| Key | Action |
|-----|--------|
| **Q** | Open quick menu (spotlights, convoy markers, FCS settings, restocking, etc.) |
| **N** | Toggle gunner IR spotlight (NV vehicles) |
| **Space** | Hunter-Killer: slew turret to TC-designated target |

---

### 🗺️ Waypoints & Time
| Key | Action |
|-----|--------|
| **F4** | Set waypoint for current vehicle |
| **F5** | Add additional waypoint |
| **Ctrl + F4** | Set direct waypoint (no pathfinding) |
| **+ / −** | Adjust time scale |
| **, / .** | Cycle time of day (dev cheat) |

---

### 🎬 Misc / HUD
| Key | Action |
|-----|--------|
| **Ctrl + F1** | Toggle HUD (debug/cinematic use — may break mouse interactions) |
| **F1 (hold)** | Show in-game controls reference |

---

Keybindings can be adjusted from the main menu or in-game at any time.
