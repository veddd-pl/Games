Arma 3 is notorious for using almost every key on a standard keyboard (and some combinations you didn't know existed). It plays more like a simulator than a standard shooter.

Here are the most important default keybinds categorized by how you'll actually use them.

---

## 🏃 Movement & Stance
Arma uses a "tactical pace" system rather than just a simple sprint.

| Action | Keybind |
| :--- | :--- |
| **Walk / Run Toggle** | `W` + `S` (Simultaneously) |
| **Sprint** | Left `Shift` (Hold) |
| **Tactical Pace** | `C` (Slower but weapon is ready) |
| **Adjust Stance** | `Ctrl` + `W` (Step up) or `S` (Step down) |
| **Prone / Crouch / Stand** | `Z` / `X` / `C` |
| **Combat Pace Toggle** | Left `Ctrl` (Double Tap) |
| **Free Look** | Left `Alt` (Hold) — *Crucial for situational awareness* |

---

## 🔫 Combat & Interaction
| Action | Keybind |
| :--- | :--- |
| **Interact / Action Menu** | `Scroll Wheel` or `Space` |
| **Main Action (Open Door/Get In)** | `Space` |
| **Inventory** | `I` |
| **Optics (Scope)** | `Right Click` (Click once) |
| **Hold Breath (Steady Aim)** | `Right Click` (Hold) |
| **Switch Weapon / Fire Mode** | `F` |
| **Reload** | `R` |
| **Throw Grenade** | `G` (*Careful! Double-tap G is the default for some mods*) |
| **Compass / Watch / Map** | `K` / `O` / `M` |

---

## 🚁 Vehicle & Pilot Controls
*Note: Helicopters in Arma 3 use a specialized physics model.*

| Action | Keybind |
| :--- | :--- |
| **Engine On/Off** | `I` (While in driver seat) |
| **Get Out** | `V` (Twice if moving) |
| **Collective Raise (Lift)** | `Left Shift` |
| **Collective Lower (Descend)** | `Left Ctrl` |
| **Auto-Hover Toggle** | `X` |
| **Flaps Up / Down** | `F` / `R` (Planes) |
| **Landing Gear** | `G` |

---

## 🗣️ Communication (Multiplayer)
If you are playing on a server, these are the keys that keep you from getting kicked for being "deaf."

* **Push to Talk (VOIP):** `Caps Lock`
* **Switch Channel:** `,` (Comma) and `.` (Period)
    * *Blue (Group), Green (Vehicle), Yellow (Side), White (Direct/Local).*
* **Chat:** `Enter`

---

## 💡 Pro Tips for New Players
1.  **The Windows Key:** In many popular mods (like ACE3 or Life), the **Left Windows Key** is the "Interaction Key" for fixing vehicles or treating wounds.
2.  **Double-Tap is Different:** Many keys behave differently if double-tapped (e.g., double-tap `Left Ctrl` to lower your weapon so you run faster).
3.  **The Pause Menu:** If you get stuck in a weird stance, pressing `Esc` often resets your character's animation state.
